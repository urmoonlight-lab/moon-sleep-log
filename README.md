# Moon Sleep Log · 月相睡眠日志

一个中文优先、无需登录、本地存储的个人睡眠故障日志。它用于记录睡眠时间、睡眠中断、睡前因素、可选个人用药备注、异常事件和次日状态。

## 本地运行

```bash
npm install
npm run dev
```

生产构建：

```bash
npm run build
npm run preview
```

## 功能

- 创建、编辑、查看和删除睡眠日志
- 本地 `localStorage` 持久化，无后端、账号或分析追踪
- 最近七天的睡眠时长、质量和常见记录因素概览
- 按日期、标签或内容搜索历史记录
- 完整 JSON / CSV 导出
- 完全省略个人用药备注字段的隐去版 JSON / CSV 导出
- JSON 备份导入
- 清除全部本地数据（带二次确认）
- 响应式移动布局与基础 PWA 离线壳

## 数据与隐私

浏览器数据保存在 `moon-sleep-log:v1` 这个 localStorage 键中。应用本身不发起数据上传；生产构建会注册一个仅用于缓存应用静态资源的 Service Worker。

清除站点数据或更换浏览器可能移除本地记录，请使用导出功能自行备份。隐去版导出会从文件中完全删除 `medicationNote` 字段或 CSV 列。

## 使用边界

Moon Sleep Log 仅用于个人记录，不提供医疗诊断、治疗、处方、剂量调整或用药改变建议，也不会分析个人用药备注。若出现异常反应、记忆空白、受伤或次日严重功能受影响，请咨询持证临床医生或药师。
